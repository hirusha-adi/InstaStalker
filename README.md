# How to install

## Linux

```bash
sudo apt update
sudo apt install python3-pip -y
pip3 install -r requirements.txt

# to run the program
python3 InstaStalker.py

# to compile the script
chmod +x compile.sh
./compile.sh
./dist/InstaStalker # running the compiled version
```
